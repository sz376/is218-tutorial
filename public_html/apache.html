How to make a start an Apache server on Docker

  1. Download docker from docker.com
  2. Enter terminal and enter your project directory.
  3. Enter touch Dockerfile if it does not already exist, then vi Dockerfile A Dockerfile is a text document that contains all the commands a user could call on the command line to assemble an image
  4. Make sure your Dockerfile has the following text:
    5. FROM httpd:2.4
    COPY ./public_html/ /usr/local/apache2/htdocs/
  5. Save and exit vi.
  6. In the terminal, enter the following docker build -t my-apache .
  7. Enter docker run -dit -p 8080:80 --name my-running-app my-apache
  8. Now open up your favorite browser and goto this url localhost:8080 to see your site.
