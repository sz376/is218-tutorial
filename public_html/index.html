How to make a git repository

  1. Enter terminal and enter “cd Desktop”
  2. Enter “mkdir” command and name it IS-Website, then “cd IS-Website”
  3. Once inside the directory, enter git init to initialize a local git repository
  4. Enter “mkdir public_html”
  5. Enter "touch public_html/index.html" to make an html page
  6. Go on to GitHub.com, log in, and make a new git repository without the README.md
  7. Copy and paste your new github link
  8. Enter "git branch -M main" to change branch name to default branch name on public repo
  9. Enter "git remote add origin https://github.com/{username}/{git_repo_name}.git" so that your changes will go to this public repo
  10. Enter git status to look at what state your files are in (changed, ready to commit, etc.)
  11. Enter "git add ." to add all files to be ready to commited
  12. Enter a "git commit -m "Intial commit" for your first commit to the repo
  13. Enter "git push origin main" to push changes to git repo
