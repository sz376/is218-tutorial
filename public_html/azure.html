How to host a website on Azure Cloud

How to setup a free student Azure account

  1. You will need a Microsoft Outlook or Hotmail email address and an academic email account to register with the program.
  2. If you don’t have a Microsoft account, create one first at https://signup.live.com/
  3. After you’ve created your email address, go to: https://azure.microsoft.com/en-us/free/students/ to join the Azure for Students plan.

How to create an Azure container

  1. If you don’t have a Microsoft account, create one first at https://signup.live.com/
  2. After you’ve created your email address, go to: https://azure.microsoft.com/en-us/free/students/ to join the Azure for Students plan.
  3. Run the following command in your terminal to log into Azure: docker login azure. This opens your web browser and prompts you to enter your Azure login credentials. If the Docker CLI cannot open a browser, it will fall back to the Azure device code flow and lets you connect manually.
  4. After you have logged in, you need to create a Docker context associated with ACI to deploy containers in ACI. Creating an ACI context requires an Azure subscription, a resource group, and a region. For example, let us create a new context called myacicontext by running docker context create aci myacicontext
  5. The command above automatically uses your Azure login credentials to identify your subscription IDs and resource groups. You can then interactively select the subscription and group that you would like to use. If you prefer, you can specify these options in the CLI using the following flags: --subscription-id, --resource-group, and --location.
  6. Now we will push our Docker image onto Docker Hub.

How to push a Docker image onto Docker Hub.

  1. Sign up for a Docker Hub account at: https://www.docker.com
  2. Sign into Docker Hub at https://hub.docker.com
  3. Click the Create Repository button. Make sure the Visibility is Public.
  4. Click the Create button!
  5. Make sure your image is built. Use the command docker build -t YOUR-IMAGE-NAME . to build the image.
  6. Login to the Docker Hub using the command docker login -u YOUR-USER-NAME
  7. Use the docker tag command to give your image a new name. Be sure to swap out YOUR-USER-NAME with your Docker ID.
    8. docker tag YOUR-IMAGE YOUR-USER-NAME/YOUR-DOCKERHUB-REPO-NAME
  8. Now use the following command to push. If you don’t specify a tag, Docker will use a tag called latest.
    9. docker push YOUR-USER-NAME/YOUR-DOCKERHUB-REPO-NAME

Deploying the Docker Image to Azure Cloud

  1. Run the following command to deploy the Docker Image to Azure:
    2. docker --context myacicontext run -p 80:80 -d YOUR-USER-NAME/YOUR-DOCKERHUB-REPO-NAME
  2. If everything went well you should be able to see your image running with the command docker --context myacicontext ps
  3. The result of the ps command will display in column “PORTS” the IP address and port on which the container is running. For example, it may show 52.154.202.35:80->80/tcp, and you can view your website page by browsing http://52.154.202.35.
  4. If your webpage does not immediately appear, give Azure a couple of minutes.
