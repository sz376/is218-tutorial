How to host a website on Azure Cloud

How to setup a free student Azure account

  1. You will need a Microsoft Outlook or Hotmail email address and an academic email account to register with the program.
  2. If you don’t have a Microsoft account, create one first at https://signup.live.com/
  3. After you’ve created your email address, go to: https://azure.microsoft.com/en-us/free/students/ to join the Azure for Students plan.

How to create an Azure container

  1. Once you have an account, navigate to https://portal.azure.com/ and click on "Create a resource"

  2. On the left hand column, select "Containers" and then "Create" under "Container Registery."

  3. Fill out the Project Details. You can create a new Resource group, for example "is218". Name it however you like after you are done, click Review + create tab and then press "Create" at the bottom.

  4. Click on "Go to resource" and then click on Access keys on the left column. Make sure admin user is set to Enabled.

  5. Open the cloud shell by clicking on the icon highlighted with the red arrow below:

  6. If an existing storage account doesn’t exist, a new one must be created.

  7. Clone the source code from GitHub to Azure: git clone https://github.com/{github username}/{repo name}
  8. Move into the directory with cd {new folder from clone}
  9. Run the build command by running az acr build --registry {Your Registry Name} --image image .

  10. Navigate back to Home, and select Create a resource. In the left column, select "Web" and then "Create" under "Web App."

  11. Fill out this details page again, make sure "Publish" is set to "Docker Container." Click on "Next: Docker" at the bottom.

  12. On this page set "Image Source" as "Azure Container Registry." and choose the registery we created on the previous steps.

  13. Click "Review + create." then "Create"
  14. Navigate back to home, you should see your new website under "Recent Resources." if not, click on "App Services."

  15. Open the resource. The URL of your website should be in the "Overview" panel. If it doesnt appear immediately, give it a couple of minutes and go back to the home screen to check again.

  16. Our site should be live at https://is218websitetutorial.azurewebsites.net
  17. To update your site after making changes to your github repo, open the cloud shell in azure like before, cd into your project directory, and git pull origin {master branch}.
  18. Enter this line again into the shell: az acr build --registry {Your Registry Name} --image image .
  19. Go back to your website resource page and the link should be updated
  20. If you are not seeing the css changes on your screen (e.g., changing your background color), you need to hard refresh your page
