How to setup a free student Azure account

  1. You will need a Microsoft Outlook or Hotmail email address and an academic email account to register with the program.
  2. If you don’t have a Microsoft account, create one first at https://signup.live.com/
  3. After you’ve created your email address, go to: https://azure.microsoft.com/en-us/free/students/ to join the Azure for Students plan.

How to create an Azure container

  1. Once you log into Azure, navigate to https://portal.azure.com/
  2. If you don’t have a Microsoft account, create one first at https://signup.live.com/
  3. After you’ve created your email address, go to: https://azure.microsoft.com/en-us/free/students/ to join the Azure for Students plan.
